> **版本**: v2.0 | **设计体系**: Glassmorphism Minimal | **DFII Score**: 16/Excellent

## 一、设计理念

### 1.1 核心美学方向

**命名**: Glassmorphism Minimal — 浅色玻璃拟态极简主义

**关键词**: 克制 · 平台感 · 轻内容 · 天空感

**差异化锚点**:

> "如果截图后去掉 Logo，如何让人认出这是爱点评？"
> 答案：**浅蓝色天空底 + 深蓝主按钮 + 白色半透玻璃卡片的组合**

### 1.2 参考方向

| 参考对象 | 学习点 |
|---------|--------|
| Linear / Vercel | 克制感、排版节奏 |
| Notion | 整洁、信息优先级 |
| Apple 官网 | 留白、比例、字重 |
| 玻璃拟态趋势 | 半透明、深度层次 |

**避免做成**:
- 传统企业官网（沉闷）
- 过度 AI 模板页（俗气）
- 紫色渐变 SaaS（通用）
- 深色模式主导（不适合内容浏览）

### 1.3 DFII 评估

| 维度 | 评分 | 说明 |
|------|------|------|
| Aesthetic Impact | 4 | 玻璃拟态已有辨识度 |
| Context Fit | 5 | AI 工具目录，用户需要快速浏览 |
| Implementation Feasibility | 5 | Tailwind 可完整实现 |
| Performance Safety | 4 | 毛玻璃需控制用量 |
| Consistency Risk | 2 | 需完整 token 系统保障 |
| **DFII Total** | **16** | **Excellent** |

## 三、玻璃拟态系统

### 3.1 核心参数

```css
/* 标准玻璃卡片 */
.glass-card {
  background: rgba(255, 255, 255, 0.25 ~ 0.5);
  backdrop-filter: blur(18px ~ 30px) saturate(140% ~ 150%);
  border: 1px solid rgba(255, 255, 255, 0.22 ~ 0.28);
  box-shadow:
    0 18px 40px rgba(15, 23, 42, 0.08),    /* 柔和弥散阴影 */
    inset 0 1px 0 rgba(255, 255, 255, 0.4); /* 内发光高光 */
}

/* 搜索面板 - 更强模糊 */
.search-panel {
  backdrop-filter: blur(30px) saturate(160%);
  background: rgba(255, 255, 255, 0.4);
}

/* 玻璃变体按重要性递减 */
.glass-card-highest { background: rgba(255, 255, 255, 0.5); }
.glass-card-high    { background: rgba(255, 255, 255, 0.35); }
.glass-card-medium  { background: rgba(255, 255, 255, 0.25); }
.glass-card-low     { background: rgba(255, 255, 255, 0.16); }
```

### 3.2 玻璃层级使用规范

| 场景 | 透明度 | blur | 使用场景 |
|------|--------|------|----------|
| 最高级 | 0.5 | 30px | Hero 主面板、搜索框 |
| 高级 | 0.35 | 24px | 主要卡片、导航栏 |
| 中级 | 0.25 | 18px | 次要卡片、内容块 |
| 低级 | 0.16 | 12px | 背景装饰、嵌套元素 |

## 四、排版系统

### 4.1 字体选择

**设计字体系统**:

| 用途 | 字体 | 备选 | 说明 |
|------|------|------|------|
| 标题/Display | Inter | system-ui | 600 字重，tight 行高 |
| 正文/Body | Inter | system-ui | 400-500 字重 |
| 代码/数据 | JetBrains Mono | monospace | 等宽，用于数字、标签 |

> **注意**: 项目使用 Next.js + Tailwind，Inter 作为默认 sans-serif 已内置。如需更独特风格可引入：
> - Display: `Outfit` (几何感)
> - Body: `Source Sans 3` (易读性)

### 4.2 字体规格表

| 层级 | 字号 (响应式) | 字重 | 行高 | 使用场景 |
|------|---------------|------|------|----------|
| h1 首页 | `text-4xl md:text-6xl` | 600 | tight | 首页大标题 |
| h1 内页 | `text-3xl md:text-5xl` | 600 | tight | 内页标题 |
| h2 区块 | `text-2xl md:text-3xl` | 600 | tight | 区块标题 |
| h3 卡片 | `text-base font-semibold` | 600 | normal | 卡片标题 |
| 正文 | `text-sm md:text-base` | 400 | 6~7 | 正文内容 |
| 标签/小字 | `text-xs md:text-sm` | 400~500 | tight | 标签、元数据 |

### 4.3 文字层次原则

```
主标题 (h1):  深蓝近黑 #11203b / #0f172a，字重 600
次级标题 (h2): slate-800，字重 600
卡片标题 (h3): slate-900，字重 600
正文内容:      slate-600，字重 400
辅助文字:      slate-500，字重 400
```

### 7.1 组件基类

| 类名 | 用途 | 核心特征 |
|------|------|----------|
| `page-shell` | 页面外壳 | 背景渐变、最小高度、滚动控制 |
| `site-header` | 导航头部 | 粘性定位、玻璃态、Logo + 导航 + CTA |
| `search-panel` | 搜索面板 | 强模糊、光影效果、圆角 32px |

| `btn-primary` | 主按钮 | 深蓝背景、白色文字、hover 加深 |
| `btn-secondary` | 次要按钮 | 玻璃背景、深蓝文字、边框 |
| `tag-semantic` | 语义标签 | 圆角-full、对应背景色、适中 padding |

### 7.2 卡片组件规范 (ToolCard)

```tsx
// 典型工具卡片结构
<article className="
  glass-card-high     // 背景 rgba(255,255,255,0.35)
  rounded-[28px]      // 大圆角
  p-5                 // 内边距 20px
  transition-all      // 过渡动画
  hover:-translate-y-0.5  // 悬浮上移（克制）
  hover:shadow-xl    // 阴影增强
">
  {/* 卡片内容 */}
</article>
```

### 7.3 悬浮交互规范

| 交互类型 | 参数 | 说明 |
|----------|------|------|
| Y轴移动 | `-translate-y-0.5` (2px) | 克制，不过度弹跳 |
| 阴影增强 | `shadow-xl` | 适度加深 |
| 缩放 | **禁止** | 保持原尺寸 |
| 背景变亮 | `brightness-105` | 可选 |

### 7.4 过渡动画规范

```css
/* 全站统一过渡 */
* {
  transition-duration: 160ms ~ 180ms;
  transition-timing-function: ease-in-out;
}

/* 特殊情况 */
.transition-fast    { transition-duration: 120ms; }
.transition-normal  { transition-duration: 160ms; }
.transition-slow    { transition-duration: 240ms; }
```

## 九、动效哲学

### 9.1 动效原则

1. **克制**: 动效服务于反馈，不装饰
2. **快速**: 160~180ms，不拖沓
3. **有意义**: 每个动画都有明确目的

### 9.2 允许的动效

| 类型 | 场景 | 参数 |
|------|------|------|
| 悬浮反馈 | 卡片、按钮 | Y轴上移 2px，阴影增强 |
| 焦点指示 | 输入框、按钮 | ring 外框 |
| 加载占位 | 骨架屏 | pulse 动画 |
| 展开/收起 | 折叠面板 | height 过渡 |

### 9.3 禁止的动效

❌ 大面积渐变流动
❌ 刺眼发光效果
❌ 每张卡片周围的径向光斑
❌ 弹跳/过冲动画
❌ 装饰性微动效

### 9.4 无障碍动效

```css
@media (prefers-reduced-motion: reduce) {
  * {
    animation-duration: 0.01ms !important;
    transition-duration: 0.01ms !important;
  }
}
```

## 十二、Anti-Patterns (必须避免)

❌ **字体**: Inter 以外的 AI 默认字体 (Roboto, Arial, OpenSans)
❌ **配色**: 紫色渐变主导、过度饱和色彩
❌ **布局**: 完全对称的卡片网格、均等的间距
❌ **动效**: 弹跳、发光、流动渐变
❌ **卡片**: 每张卡片加径向光斑装饰
❌ **对比**: 深色模式作为默认主题

## 十三、设计验证清单

发布前逐项检查:

- [ ] 只有一个主色（深蓝 #11203b）

- [ ] 卡片使用玻璃态而非实心白色
- [ ] 圆角统一使用定义的值
- [ ] 悬浮效果克制（≤2px 上移）
- [ ] 动效时长 160~180ms

- [ ] 对比度通过 WCAG AA
- [ ] 支持 `prefers-reduced-motion`
- [ ] 首屏有冲击力但不拥挤
