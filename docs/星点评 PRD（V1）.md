# AI 工具集合网站 PRD（V1）

## 1. 项目概述

### 1.1 项目名称

星点评V1（AI 工具集合平台）

### 1.2 项目背景

当前市面上 AI 工具数量快速增长，用户在寻找合适工具时常面临以下问题：

- 工具分散，缺少统一入口

- 工具分类不清晰，筛选成本高

- 缺少工具详情、评价和真实使用反馈

- 收藏和管理自己感兴趣工具不方便

因此，需要设计一个 **AI 工具集合网站**，帮助用户快速搜索、筛选、浏览和收藏 AI 工具，并通过评论系统形成用户反馈闭环。

### 1.3 项目目标

构建一个面向普通用户的 AI 工具导航与发现平台，支持：

- 以极简、高级感首页提升品牌辨识度与点击效率

- 快速搜索 AI 工具

- 按类别和标签筛选工具

- 通过首页专题模块进入对应“更多”专题页

- 查看工具详情与评论

- 提交评分与评论

- 查看同类工具推荐

- 收藏感兴趣工具

- 进入个人中心管理收藏与评论记录

- 通过内部后台维护工具、分类、标签和展示状态

---

# 2. 产品定位

### 2.1 产品定位

一个聚合 AI 工具、支持筛选、收藏、评论和个人管理的工具发现平台。

### 2.1.1 视觉方向

- 首页整体风格参考 aibot 一类的极简导航站，但视觉质量需要更高

- 风格关键词：克制、轻量、专业、留白充足、卡片信息密度高但不拥挤

- 设计目标不是“像导航站一样堆链接”，而是“像精选目录一样做内容编排”

### 2.2 目标用户

主要用户包括：

- 想寻找 AI 写作、AI 视频、AI 编程等工具的普通用户

- 希望快速比较和发现新工具的内容创作者

- 关注效率工具的职场用户

- 对 AI 产品有探索兴趣的学生和开发者

### 2.3 核心价值

- 降低用户寻找 AI 工具的时间成本

- 提供清晰分类和标签体系

- 通过评论帮助用户判断工具质量

- 通过收藏和个人中心提升用户留存

# 3. 产品范围

## 3.1 本期范围

本期包含 6 个核心界面：

1. 首页

2. 工具列表页

3. 专题更多页

4. 工具详情页

5. 个人中心页

6. 内部管理后台

## 3.2 暂不包含

以下功能暂不纳入 V1：

- 工具对比

- 多语言切换

- 支付体系

- 面向外部用户的工具提交平台

- 复杂审核流与多级审批

- 私信/社交功能

- 消息通知中心

- 推荐算法个性化排序

# 4. 信息架构

## 4.1 页面结构

- 首页

- 工具列表页

- 专题更多页

- 工具详情页

- 个人中心页

- 内部管理后台

## 4.2 页面关系

- 用户进入首页，优先浏览精选工具和专题模块

- 用户可从首页搜索进入列表页，或点击专题模块下的“更多”进入专题更多页

- 用户在列表页与专题更多页中查看完整结果并进入工具详情页

- 点击某个工具卡片，进入工具详情页，可查看评分、评论和相关推荐

- 点击左侧头像区域，进入个人中心页

- 在个人中心中可查看收藏工具和评论过的内容，并支持退出登录与注销账号

- 内部运营人员通过后台维护工具信息、分类标签、专题归属和上下线状态

# 5. 功能需求

---

## 5.1 首页

### 5.1.1 页面目标

用极简但高质量的方式展示站点价值、热门工具与专题入口，帮助用户快速开始浏览。

### 5.1.2 页面布局

页面整体采用上中下结构：

- 顶部：品牌区 + 搜索框 + 关键导航入口

- 中部：首屏推荐区 / 热门工具区 / 专题模块区

- 底部：补充分类入口、友情链接或平台说明
### 5.1.3 功能模块

#### 模块 1：顶部搜索框

**功能说明**

- 用户可通过关键词搜索 AI 工具

- 支持按工具名称、标签、用途进行模糊搜索

**交互说明**

- 用户输入关键词后点击搜索 / 回车触发搜索

- 搜索结果刷新右侧工具列表

**输入示例**

- AI 写作

- 视频生成

- 编程助手

---

#### 模块 2：首屏推荐区

**功能说明**  
展示少量精选 AI 工具，强调高质量和高点击率，不追求堆量。

**展示内容**

- 头部主推工具 1~3 个

- 工具名称、简介、核心标签、官网按钮

- 可进入详情页

**交互说明**

- 点击工具卡片进入详情页

- 点击官网按钮新窗口打开官网

---

#### 模块 3：专题模块区

**功能说明**  
按场景或类别组织专题内容，例如 AI 写作、AI PPT、AI 视频等。

**模块内容**

- 模块标题

- 该专题下精选工具卡片

- 模块底部 `>> 更多AI工具`

**交互说明**

- 点击卡片进入详情页

- 点击 `>> 更多AI工具` 进入对应专题更多页

---

**交互说明**

- 用户输入关键词后点击搜索 / 回车触发搜索

## 5.3 专题更多页

### 5.3.1 页面目标

承接首页专题模块流量，展示某一主题下更完整、更聚焦的工具集合。

### 5.3.2 页面定义

“更多页”不是独立的泛工具库，而是首页每个专题模块对应的二级聚合页，例如：

- AI 写作工具更多页

- AI PPT 工具更多页

- AI 视频工具更多页

### 5.3.3 页面结构

- 专题头图 / 标题 / 简要说明

- 专题下的推荐工具列表

- 排序与标签筛选区

- 分页或加载更多

### 5.3.4 交互说明

- 默认按专题相关度 + 热度排序

- 支持继续按标签筛选

- 支持进入详情页与跳转官网

- 页面保留返回首页专题模块的路径感

## 5.4 工具详情页

### 5.4.1 页面目标

帮助用户全面了解某个 AI 工具，并查看他人反馈、提交自己的评论。

### 5.4.2 页面核心信息

详情页需展示以下内容：

- 工具名称

- 主要用途

- 跳转官网按钮

- 评论区

- 查看他人评论

- 同类工具推荐

### 5.4.3 功能模块

#### 模块 1：工具基础信息

**展示内容**

- 工具名称

- 工具 Logo

- 简介

- 主要用途

- 标签

- 官网跳转按钮

**交互说明**

- 点击官网按钮，跳转至该工具官网

---

#### 模块 2：同类工具推荐

**功能说明**  
在工具详情页中，基于当前工具所属类别、标签或主要用途，推荐若干相似工具，方便用户横向浏览与比较。

**推荐逻辑建议**  
优先按以下规则推荐：

1. 同一工具分类
2. 相同或相近标签
3. 相似使用场景 / 主要用途
4. 热度较高或评价较好的工具优先展示

**展示位置建议**

- 放在工具基础信息下方、评论区上方  
  或
- 放在页面右侧作为推荐区域

V1 更建议放在 **详情主体中下部，评论区上方**，更自然。

**展示形式**  
以卡片或横向列表形式展示，每个推荐工具包含：

- 工具名称
- Logo
- 一句话简介
- 标签
- 跳转详情入口
- 官网跳转按钮（可选）

**推荐数量建议**

- 默认展示 3~6 个

**交互说明**

- 点击推荐卡片进入对应工具详情页
- 点击官网按钮跳转官网
- 当前工具本身不可出现在推荐列表中

**空状态**

- 若暂无同类工具，则不展示该模块
- 或显示提示：`暂无更多同类工具推荐`

---

#### 模块 3：评分与评论区

**功能说明**  
用户可对该工具进行评价和评论。

**评论内容建议包含**

- 评分（1-5 分）

- 评论内容

- 评论时间

- 用户昵称 / 头像

- 已评分用户的历史评分

**交互说明**

- 已登录用户可发表评论

- 未登录用户点击评论时，提示登录

- 提交后评论显示在评论列表中

- 一个用户对同一工具仅保留一条评论，后续提交视为覆盖更新

---

#### 模块 4：他人评论列表

**功能说明**  
展示其他用户对该工具的评论内容，帮助用户判断工具质量。

**展示内容**

- 用户头像

- 用户昵称

- 评论时间

- 评论正文

- 评分（如有）

**排序方式**

- 默认按最新排序

- 支持按评分高低排序

- 后续可扩展按最有帮助排序

## 5.6 内部管理后台

### 5.6.1 页面目标

为内部运营提供够用、直接、低成本的工具维护能力，不追求复杂流程设计。

### 5.6.2 角色说明

- 管理员：可查看与编辑全部工具数据

- 运营人员：可新增、修改、上下架工具，可维护分类、标签和专题归属

### 5.6.3 核心模块

#### 模块 1：工具管理

- 新增工具

- 编辑工具基础信息

- 上下架工具

- 设置是否首页推荐

- 设置所属专题与分类

#### 模块 2：分类与标签管理

- 新增、编辑、删除分类

- 新增、编辑、删除标签

- 维护专题与工具映射关系

#### 模块 3：评论管理

- 查看评论列表

- 删除明显异常评论

- 简单标记处理状态

#### 模块 4：轻量审核

- 工具数据保留 `draft`、`published`、`archived` 等简单状态

- 不做复杂审批链路，仅用于内部“先录入再发布”

# 6. 用户流程

## 6.1 浏览工具流程

1. 用户进入首页

2. 浏览首页精选工具和专题模块

3. 点击专题模块中的 `>> 更多AI工具` 进入专题更多页，或通过顶部搜索进入列表页

4. 在列表页 / 专题更多页使用“热度 / 新推出 / 标签”进一步筛选

5. 点击某个工具进入详情页

6. 查看详情、评分、评论和相关推荐

## 6.2 收藏流程

1. 用户在首页工具卡片点击收藏

2. 若未登录，提示登录

3. 登录后收藏成功

4. 收藏内容同步到个人中心“收藏工具”

## 6.3 评论流程

1. 用户进入工具详情页

2. 查看其他人评论

3. 登录后提交自己的评论

4. 评论成功后展示在评论列表

5. 个人中心“评论过的内容”中同步显示

## 6.4 个人中心流程

1. 用户点击左侧头像进入个人中心

2. 查看头像和账号信息

3. 进入收藏工具二级菜单

4. 进入评论过的内容二级菜单

5. 可退出登录或注销账号

# 8. 交互与状态说明

## 8.1 登录状态

- 未登录时：
  
  - 可浏览工具
  
  - 不可收藏
  
  - 不可评论
  
  - 点击头像进入登录流程

- 已登录时：
  
  - 可收藏工具
  
  - 可评论
  
  - 可进入个人中心

## 8.2 空状态

需考虑以下空状态页面：

- 搜索无结果

- 某分类下暂无工具

- 用户暂无收藏

- 用户暂无评论记录

## 8.3 错误状态

需考虑：

- 首页专题推荐加载失败

- 工具列表加载失败

- 详情加载失败

- 评论提交失败
